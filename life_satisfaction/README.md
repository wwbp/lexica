# Life Satisfaction


This archive contains the Life Satisfaction lexicon, derived from 2,143 participants who shared Facebook statuses and the life satisfaction measure Cantrilâ€™s Ladder. Please see the original [paper](https://www.pnas.org/content/117/19/10165) for more details.

To learn how to use this lexicon, please refer to the [paper](http://wwbp.org/publications.html#p83). 

Note that the intercept (marked as "\_intercept" in the lexica csvs) is to be added post weighted lexicon extraction.
